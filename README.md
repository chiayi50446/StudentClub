# Student Club

## Description

- use MERN
- use nodemon to restart server when file is changed
- use .env file to manage config and hide sensitive information

## Install

### Step1

Edit file's name from `.env.example` to `.env`

### Step2

Set mongodb connect string in `.env`

### Step3

Install modules

> npm install<br>
> cd client<br>
> npm install

## Run

> cd client<br>
> npm run dev<br>
